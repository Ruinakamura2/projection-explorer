# 投影変換ビジュアライザ

3次元空間から2次元平面への様々な投影関数を、座標グリッド・座標軸・ボックスでインタラクティブに可視化するツールです。`index.html` をブラウザで開くだけで動作します。

## 投影モード

| モード | 説明 |
|--------|------|
| 正距方位 (r∝θ) | 角度に比例した半径。一般的な魚眼レンズ |
| 正射影 (r∝sinθ) | sin に比例。FOV=180° 固定 |
| 透視図法 (gnomonic) | 直線が直線に写る。標準的なカメラ投影 |
| λ-tan ブレンド | 正距方位と透視の中間をブレンド |
| メビウス変換 | 複素数によるメビウス変換をレンズに適用 |
| 万華鏡 | 鏡像反復による万華鏡パターン |
| カスタム g | swirl・ripple・fold パラメータで自由に歪める |
| ステレオグラフィック | 球面の等角写像。円が円に写る |
| 逆透視 | 前方が外周・後方が中心の内外反転 |
| 複素対数 | 複素対数写像による渦状変換 |
| 等積 (Lambert) | 面積を保存する Lambert 等積射影 |
| 冪乗族 | 指数 p で中心と周辺の圧縮率を調整 |
| 双曲線 (tanh) | tanh による中心集中型の圧縮 |
| 複素冪 | 複素数の冪乗による回転・拡大変換 |
| 波紋 | 外向きに波面を送り続けるアニメーション写像 |

## カメラ操作

| キー | 動作 |
|------|------|
| W / S | 前後移動 |
| A / D | 左右移動 |
| Q / E | 上下移動 |
| ↑ / ↓ | 視点を上下に向ける |
| ← / → | 視点を左右に向ける |
